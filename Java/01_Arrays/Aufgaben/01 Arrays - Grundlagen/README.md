# Arrays - Grundlage
 *Bei Fragen mir eine Nachricht senden!*

## Was du lernen wirst:
- Wie ein Array aufgebaut ist
- Methoden, um mit einem Array zu arbeiten
    - Hinzufügen von Objekten
    - Löschen von Objekten
    - Kombinieren von Arrays
    - Sortieren von Arrays

## Aufgabenstellung:
1. Erstellen ein Array mit dem Datentyp String mit der Länge 3.
2. Speichere 3 Vornamen deiner Wahl im Array.
3. Erstelle ein weiteres Array mit dem Datentyp String und der Länge 3.
4. Speichere 3 Nachnamen deiner Wahl im Array.
5. Erstelle nocheinmal ein Array mit der Länge 3 und speichere jeweils Vorname und Nachname der beiden Arrays davor per Index ab. 
6. Lösche das Element am Index 0 des kombinierten Arrays
7. Erstelle wieder ein Array mit dem Datentyp int und der Länge 10.
8. Belege das Array mit random Zahlen (upper limit = 100).
9. Sortiere das Array absteigend.
10. Erstelle einen int mit der Summe aller Zahlen aus dem Array
